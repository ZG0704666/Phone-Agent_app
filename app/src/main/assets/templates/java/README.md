# Operit Java 项目

这是一个使用 Operit 创建的 Java 项目。

## 快速开始

使用 Gradle:
1. 点击 "gradle build" 构建项目
2. 点击 "gradle run" 运行程序

使用 Maven:
1. 点击 "mvn clean install" 构建项目
2. 运行编译后的 JAR 文件

## 项目结构

- `Main.java` - 主程序入口
- `.operit/config.json` - Operit 工作区配置

## 自定义

您可以：
- 添加更多 Java 类和包
- 创建 `build.gradle` 或 `pom.xml` 配置文件
- 在 `.operit/config.json` 中自定义构建命令
